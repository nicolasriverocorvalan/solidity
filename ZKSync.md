## ZKSync cli

### Create a local node

```
npm install -g zksync-cli
npm update -g zksync-cli
npx zksync-cli dev config (In memory node - Quick startup, no persisted state, only L2 node - zkcli-in-memory-node)
npx zksync-cli dev start
```
